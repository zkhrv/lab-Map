1. Метод Node.__init__: создает новый узел дерева с заданным ключом, данными и указателями на 
предыдущий, левый и правый узлы.

2. Метод Map.__init__: инициализирует пустое дерево. Если передан аргумент map, то создает новое дерево, 
которое является копией переданного дерева.

3. Метод Map.copy_tree: создает новое дерево, которое является копией переданного дерева.

4. Метод _add_node(: добавляет новый узел в дерево с заданным ключом и данными. Если узел с таким ключом уже 
существует в дереве, выводится соответствующее сообщение.

5. Метод add_element(: добавляет новый узел в дерево с заданным ключом и данными.

6. Метод clear: очищает дерево.

7. Метод _clear_tree: рекурсивно очищает дерево.

8. Метод get_element_node: находит узел дерева с заданным ключом и возвращает его. Если узел с 
таким ключом не найден, выводится соответствующее сообщение.

9. Метод change_element: изменяет данные узла с заданным ключом.

10. Метод size: выводит количество узлов в дереве.

11. less_than: Метод сравнивает ключ корневого узла текущего дерева с ключом корневого 
узла другого дерева, переданного в качестве аргумента. Возвращает булево значение True, если ключ текущего дерева 
меньше ключа другого дерева, и False в противном случае.

12. less_than_or_equal_to: Метод сравнивает ключ корневого узла текущего дерева с ключом корневого узла другого дерева, 
переданного в качестве аргумента. Возвращает булево значение True, если ключ текущего дерева меньше или равен ключу
другого дерева, и False в противном случае.

13. equal_to: Метод сравнивает ключ корневого узла текущего дерева с ключом корневого узла другого дерева,
переданного в качестве аргумента. Возвращает булево значение True, если ключ текущего дерева равен ключу другого 
дерева, и False в противном случае.

14.  not_equal_to: Метод сравнивает ключ корневого узла текущего дерева с ключом корневого узла 
другого дерева, переданного в качестве аргумента. Возвращает булево значение True, если ключ текущего дерева 
не равен ключу другого дерева, и False в противном случае.

15. greater_than: Метод сравнивает ключ корневого узла текущего дерева с ключом корневого узла другого дерева, 
переданного в качестве аргумента. Возвращает булево значение True, если ключ текущего дерева больше ключа другого 
дерева, и False в противном случае.

16. greater_than_or_equal_to: Метод сравнивает ключ корневого узла текущего дерева с ключом корневого узла другого 
дерева, переданного в качестве аргумента. Возвращает булево значение True, если ключ текущего дерева больше или 
равен ключу другого дерева, и False в противном случае.
