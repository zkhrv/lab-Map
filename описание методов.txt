__init__(self, key, data, left=None, right=None): Конструктор узла, который инициализирует ключ, данные,
 а также левого и правого потомков. Высота узла инициализируется значением 0.
Класс Дерева:

__init__(self): Конструктор класса, который инициализирует корневой узел как None и количество узлов как 0.

updateHeight(self, n): Метод, который обновляет высоту узла n на основе высот его потомков.

height(self, n): Метод, который возвращает высоту узла n. Если узел равен None, то возвращается -1.

getBalance(self, n): Метод, который возвращает баланс узла n. Если узел равен None, то возвращается 0.

rotateRight(self, y): Метод, который осуществляет поворот узла y направо.
 Метод возвращает узел x, который становится новым корнем поддерева.

rotateLeft(self, y): Метод, который осуществляет поворот узла y налево.
 Метод возвращает узел x, который становится новым корнем поддерева.

rebalance(self, z): Метод, который перебалансирует дерево относительно узла z, если он несбалансирован.
 Метод возвращает новый корень поддерева.

AddEl(self, key, data): Метод, который добавляет новый узел в дерево с ключом key и данными data.

insert(self, root, key, data): Метод, который рекурсивно добавляет новый узел с ключом key и данными
 data в дерево, начиная с корневого узла root.

clear(self): Метод, который удаляет все узлы из дерева.

ClearTree(self, root): Метод, который рекурсивно удаляет все узлы из дерева, начиная с корневого узла root.

ElemetnPoinet(self, key): Метод, который возвращает узел с ключом key из дерева.

ChangeEl(self, key, data): Метод, который изменяет данные узла с ключом key на новые данные data.

Size(self): Метод, который возвращает количество узлов в дереве.
